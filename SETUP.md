# قطة الدوام — دليل الإعداد 🍔

## المتطلبات
- Node.js v22+ (عندك v24 ✅)
- Telegram account
- Claude API key (من console.anthropic.com)

---

## 1. إعداد بوت تيليقرام

1. افتح تيليقرام وروح لـ **@BotFather**
2. أرسل `/newbot`
3. اختار اسم للبوت (مثلاً: قطة دوامي)
4. اختار username (مثلاً: `qattah_dawam_bot`)
5. احفظ الـ **token** اللي يعطيك إياه

---

## 2. إعداد Claude API

1. روح على [console.anthropic.com](https://console.anthropic.com)
2. أنشئ API key جديد
3. احفظه

---

## 3. تشغيل البوت

```bash
cd bot

# انسخ ملف الإعدادات
copy .env.example .env

# عدّل .env وأضف توكناتك:
# BOT_TOKEN=1234567890:ABCdef...
# CLAUDE_API_KEY=sk-ant-...

# شغّل البوت
start-bot.bat
```

أو من command line:
```bash
node index.js
```

ستشوف:
```
  Bot: @your_bot_username
  API Server: http://192.168.x.x:3001
  Mobile App URL: http://192.168.x.x:3001
```

---

## 4. إضافة البوت للجروب

1. افتح جروب الدوام
2. أضف البوت كعضو
3. **مهم**: اعطه صلاحية "إدارة المجموعة" أو على الأقل "إرسال رسائل"
4. **كل شخص في الجروب يرسل `/start` للبوت خاص** — عشان يقدر يرسله المبلغ بالخاص

---

## 5. إعداد معلومات الدفع

من تيليقرام (خاص مع البوت):
```
/set stcpay 0512345678
/set iban SA0012345678901234567890
/set name محمد الأحمد
```

أو من تطبيق الجوال (الإعدادات).

---

## 6. تطبيق الجوال (React Native)

### التثبيت
```bash
cd QattahMobile
npm install
npx expo start
```

- اسكان QR من تطبيق **Expo Go** (متوفر في App Store / Google Play)
- في شاشة الإعدادات، أدخل رابط السيرفر: `http://192.168.x.x:3001`

---

## طريقة الاستخدام اليومية

### في الجروب:
1. اكتب: **"نطلب من البيك"** أو **"اليوم هنقرستيشن"**
2. البوت يعرض تطبيقات التوصيل → اختار
3. البوت يبحث عن المطعم → اختار الصح
4. البوت يحمّل المنيو وينشره
5. كل شخص يكتب طلبه بالعربي: "أبغى كريسبي مع كول سلو"
6. البوت يأكد الطلب ويطلب التأكيد
7. عندما تنتهي الطلبات: اكتب **"تم الطلب"**
8. كل شخص يوصله DM بنصيبه وبيانات الدفع
9. أنت يوصلك ملخص بالكل

### أوامر مفيدة:
```
/status   — عرض الطلبات الحالية
/done     — إغلاق الطلب
/settings — الإعدادات (خاص مع البوت)
```

---

## تنبيهات مهمة

⚠️ **السكريبنق**: تطبيقات التوصيل قد تغير تصميمها. إذا البوت ما قدر يجيب المنيو، يمكن إدخاله يدوياً.

⚠️ **DM**: البوت يقدر فقط يرسل خاص للأشخاص اللي أرسلوا له `/start` أولاً.

⚠️ **الشبكة**: الجميع يجب أن يكونوا على نفس شبكة الواي فاي لتطبيق الجوال يشتغل. أو شغّل البوت على سيرفر cloud.
