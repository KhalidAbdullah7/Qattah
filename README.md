# 🍔 قطة الدوام

**حل ذكي لمشكلة الطلبات الجماعية في بيئة العمل**

بوت تيليقرام يعيش في جروب الدوام، يجيب المنيو تلقائياً، يسجّل طلب كل شخص، ويرسل له نصيبه بالخاص.

## 🎯 المشكلة اللي يحلها

> كل يوم نطلب أكل بالدوام، بس وقت الدفع ما نعرف مين طلب إيش — ونبلش نحسب يدوياً!

**قطة الدوام** يحل هذه المشكلة بالكامل:

| قبل | بعد |
|-----|-----|
| ❌ فوضى في الجروب | ✅ البوت ينظم كل شيء |
| ❌ ما تعرف مين طلب إيش | ✅ كل طلب مسجّل باسم صاحبه |
| ❌ حساب يدوي مزعج | ✅ البوت يحسب ويرسل لكل شخص نصيبه |
| ❌ إدخال المنيو يدوياً | ✅ يجيب المنيو تلقائياً من التطبيق |

## ✨ المميزات

- 🤖 **بوت تيليقرام** — يعيش في جروب الدوام ويفهم العربي

- 📱 **تطبيق جوال** — لوحة تحكم للإداري (React Native)
- 📊 **ملخص فوري** — يرسل للإداري ملخص كامل بكل الطلبات والمبالغ

## 🚀 خطوات الإعداد

### الخطوة 1: إنشاء بوت تيليقرام

1. افتح تيليقرام وروح لـ **[@BotFather](https://t.me/BotFather)**
2. أرسل `/newbot`
3. اختار اسم للبوت مثلاً: `قطة دوامي`
4. اختار username مثلاً: `qattah_dawam_bot`
5. 📋 **احفظ الـ Token** — ستحتاجه لاحقاً

### الخطوة 2: الحصول على Claude API Key

1. روح على **[console.anthropic.com](https://console.anthropic.com)**
2. أنشئ حساباً أو سجّل دخول
3. من **API Keys** → أنشئ مفتاح جديد
4. 📋 **احفظ المفتاح**

### الخطوة 3: تشغيل البوت

```bash
# 1. انسخ مجلد البوت
cd bot

# 2. أنشئ ملف الإعدادات
copy .env.example .env
```

عدّل ملف `.env`:
```env
BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrSTUvwxYZ
CLAUDE_API_KEY=sk-ant-api03-...
```

```bash
# 3. شغّل البوت
start-bot.bat
```

ستشوف:
```
╔══════════════════════════════════════════╗
║      🤖 قطة الدوام - بوت تيليقرام        ║
╚══════════════════════════════════════════╝

  Bot: @qattah_dawam_bot
  API Server: http://192.168.1.5:3001

  ✅ البوت شغال! أضفه للجروب وابدأ.
```

### الخطوة 4: إضافة البوت للجروب

1. افتح جروب الدوام في تيليقرام
2. أضف البوت كعضو (ابحث عن `@username_bot`)
3. اعطه صلاحية إرسال الرسائل
4. ⚠️ **مهم:** كل شخص في الجروب يرسل `/start` للبوت **خاص** (مرة واحدة فقط) حتى يقدر يرسله المبلغ لاحقاً

### الخطوة 5: ضبط بيانات الدفع

من خاص مع البوت:
```
/set stcpay 0512345678
/set iban SA1234567890123456789012
/set name محمد الأحمد
```

### الخطوة 6: تطبيق الجوال (اختياري)

```bash
cd QattahMobile
npm install
npx expo start
```

اسكان QR Code من تطبيق **[Expo Go](https://expo.dev/go)** على جوالك، ثم في شاشة الإعدادات أدخل:
```
http://192.168.1.5:3001   ← رابط السيرفر اللي ظهر عند تشغيل البوت
```

## 📖 طريقة الاستخدام اليومية

### في جروب تيليقرام:

**1️⃣ بدء الطلب**
```
أنت: نطلب من البيك اليوم
```
```
البوت: 🍔 رائع! طلب من البيك
        من أي تطبيق توصيل؟
        [ هنقرستيشن ] [ كيتا ] [ جاهز ]
```

**2️⃣ اختيار المطعم**

البوت يبحث ويعرض النتائج → تختار المطعم الصح → البوت يحمّل المنيو وينشره في الجروب

**3️⃣ الكل يطلب بالعربي**
```
أحمد:   أبغى كريسبي مع كول سلو
البوت:  ✅ @أحمد: دجاج كريسبي × 1 + كول سلو × 1 = 33 ريال — صح؟
        [ ✅ أيوه صح ]  [ ❌ لا، عدّل ]

محمد:   واحد بيق تشيكن
البوت:  ✅ @محمد: بيق تشيكن × 1 = 30 ريال — صح؟
```

**4️⃣ إغلاق الطلب**
```
أنت:    تم الطلب
البوت:  ✅ تم إغلاق الطلب! جاري إرسال المبالغ...
```

**5️⃣ كل شخص يستقبل رسالة خاصة**
```
🧾 طلبك من البيك:
• دجاج كريسبي × 1 = 25.00 ريال
• كول سلو × 1 = 8.00 ريال

المجموع: 33.00 ريال

💸 حوّل المبلغ على:
STC Pay: 0512345678 — محمد الأحمد
```

**6️⃣ الإداري يستقبل الملخص**
```
📊 ملخص الطلب — البيك

• أحمد: 33.00 ريال ✅
• محمد: 30.00 ريال ✅

💰 الإجمالي: 63.00 ريال
👥 2 أشخاص
```

## 🤖 أوامر البوت

| الأمر | الوظيفة | أين يُستخدم |
|-------|---------|------------|
| `/start` | تسجيل المستخدم | خاص مع البوت ✉️ |
| `/order [مطعم]` | بدء طلب جديد | الجروب 👥 |
| `/status` | عرض الطلبات الحالية | الجروب 👥 |
| `/done` | إغلاق الطلب وإرسال المبالغ | الجروب 👥 |
| `/settings` | عرض الإعدادات | خاص مع البوت ✉️ |
| `/set [مفتاح] [قيمة]` | تغيير إعداد | خاص مع البوت ✉️ |

**مفاتيح `/set`:**
```
/set stcpay  0512345678          ← رقم STC Pay
/set iban    SA00...             ← رقم IBAN
/set name    محمد الأحمد         ← اسم صاحب الحساب
/set claude  sk-ant-...          ← Claude API Key
```

## ⚠️ ملاحظات مهمة

> **السكريبنق:** تطبيقات التوصيل قد تغير تصميمها في أي وقت مما قد يؤثر على جلب المنيو. في هذه الحالة يمكن إدخال المنيو يدوياً من الواجهة الويب البديلة.

> **رسائل DM:** البوت يرسل رسائل خاصة فقط للأشخاص اللي أرسلوا `/start` له مسبقاً. تأكد أن كل أعضاء الجروب يفعلون ذلك.

> **الشبكة:** التطبيق الجوال يتصل بالسيرفر عبر الشبكة المحلية. إذا أردت وصولاً من أي مكان، يمكن نشر البوت على سيرفر cloud مثل Railway أو Render.
